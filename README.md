## 간단한 테이블 표

| 명령어 | 설명 | 예시 | 
| ----- | ---------------------------- | ---------- |
|   top   |   실시간 프로세스 모니터링   |   top   |
|   ps    |   현재 실행중인 프로세스 출력   |   ps -e   |
|   jobs  |   백그라운드/중지된 작업 표시   |   jobs -l   |
|   kill  |   프로세스 종료   |   kill   |

---

## 명령어 상세 설명


**1. top** : 현재 OS의 상태를 나타내주는 CLI 어플리케이션이다. 메모리 사용량, CPU 사용량 등을 나타내주며 top를 실행하는 동안에는 주기적인 업데이트로 실시간에 근접한 내용을 보여준다.


**2. ps** : Process Status의 약자, 현재 시스템 상에서 작동 중인 프로세스를 확인할 수 있는 명령어이다.
   - -e: 현재 사용자뿐만 아니라 다른 사용자들이 구동시킨 모든 프로세스를 보여준다.
   - -f: 보다 상세한 정보를 보여준다.
   - -l: -f보다 더 상세한 정보를 보여준다.

     
**3. jobs** : 현재 쉘 세션에서 실행 중이거나 일시중지된 작업을 확인할 때 사용된다. 백그라운드나 포그라운드에서 실행되는 작업들을 목록으로 보여주며, 각 작업에 대한 상태 정보도 함께 제공한다. 각 작업에 대해 작업 번호, 프로세스 ID, 작업 상태, 명령어 이름 등을 출력한다.
   - -l: 각 작업의 프로세스 ID도 함께 출력한다.
   - -p: 각 작업의 프로세스 ID만 출력한다.
   - -n: 가장 최근에 상태가 변경된 작업만 출력한다.
   - -r: 실행 중인 작업만 출력한다.
   - -s: 일시 중지된 작업만 출력한다.

  
**4. kill** : 특정 프로세스나 프로세스 그룹에 시그널을 보내 프로세스를 종료하는 데 사용된다.
   - 사용법 : kill [옵션] <프로세스 ID>
